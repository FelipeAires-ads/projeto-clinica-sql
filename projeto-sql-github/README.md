# Projeto Banco de Dados - Clínica Hospitalar

Este repositório contém:

- Modelo lógico e DER
- Scripts SQL (CREATE, INSERT, SELECT, UPDATE, DELETE)
- Imagens e documentação

## Estrutura
/projeto-sql-github
 ├── sql/      # Scripts SQL
 ├── docs/     # Documentação
 └── img/      # Imagens do DER

## Como executar
Você pode rodar o SQL usando:
- MySQL Workbench
- DBeaver
- SQLiteOnline
- DBFiddle
